# KPI framework

## Quality

- production-quality companies
- source-confirmed companies
- page-evidence coverage
- verified publication-date coverage
- double-review completion
- stale-verification count
- broken-source-link count
- progress-connected companies
- historical-plan-connected companies

## Product usage

- search usage rate
- strategy-filter usage rate
- comparison usage rate
- companies per comparison
- official-source click-through rate
- return usage
- zero-result search rate
- mobile completion rate

## Revenue

- spot-product sales
- qualified paid-intent users
- Pro paid conversion
- monthly recurring revenue
- corporate contracts
- annual contract value
- gross retention
- revenue retention
- API and data-license contracts

## North-star sequence

1. trusted production-quality company count
2. repeated research workflows completed
3. recurring paid users and teams
4. recurring revenue retained
