# Data review roles

## Extractor

- identifies the official source;
- records source metadata;
- extracts candidate facts and evidence pages;
- marks unresolved questions.

## Reviewer

- compares extracted facts with the original document;
- verifies values, units, periods, and context;
- checks strategy classification;
- confirms unknown and undisclosed states are used correctly.

## Approver

- performs the second review required for production promotion;
- confirms the quality profile and display;
- approves the release history entry.

A single person may perform multiple roles during the PoC, but production promotion must retain a separate second-check record.
