## 1. 現在地

Chu-keiは、日本上場企業の中期経営計画を探し、比較し、原文まで検証できる企業調査基盤である。

## 2. 今回の拡張

1. JPX公式上場銘柄一覧から600社を新規追加した。
2. 新規企業は推測で中計情報を付けず、すべてCoverageβとして登録した。
3. 新規未調査企業から100社を選定し、JPX企業コード別の公式開示を調査した。
4. 100社の結果は、eligible 78社、追加確認18社、失敗4社だった。
5. eligible 78社のうち、2024年以降の正式な中計・経営戦略、企業同一性、複数ページ証跡、構造化項目が揃う49社だけを承認した。
6. 49社に一次レビューと独立再検証を付与し、本番品質へ昇格した。

本番品質は1,500社から1,549社へ増加した。追加確認が必要な企業を件数合わせで採用していない。

## 3. 品質基準

本番品質企業には次を必須とする。

- 会社公式またはJPX企業コード別の公式一次資料
- 企業名または証券コードの一致
- 資料名、公表日、最終確認日
- PDFページ番号または具体的な公式Web見出し
- 経営課題、戦略、数値、資本政策等の構造化
- 数値の単位、年度、目標・実績区分への注意
- 比較不能な場合の明示
- 一次レビュー
- 独立再検証

自動事実補完、自動選定、自動承認、自動本番昇格は禁止する。
