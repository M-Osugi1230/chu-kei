## 4. 品質基準

本番品質企業には次を必須とする。

- 会社公式またはJPX企業コード別の公式一次資料
- 企業名または証券コードの一致
- 資料名、公表日、最終確認日
- PDFページ番号または具体的な公式Web見出し
- 経営課題、戦略、数値、資本政策等の構造化
- 数値の単位、年度、目標・実績区分への注意
- 比較不能な場合の明示
- 一次レビュー
- 独立再検証

比較不能な目標に架空の進捗率を作らない。上場廃止企業は現行上場企業数へ含めず、公式根拠付きで履歴を保存する。

## 5. 基盤改善

- 1社の複数公式資料を比較する第2世代調査エンジン
- 公表日を固定した再調査ルート
- 公式PDF全件を対象にする回復調査ルート
- 調査スクリプトを設定ファイル単位で切り替えるパイプライン
- 上場廃止企業のアーカイブと最新JPX一覧からの自動置換
- 提案SHA-256、現在ステージ、企業同一性による再昇格防止
- 一次レビューと独立再検証の分離
