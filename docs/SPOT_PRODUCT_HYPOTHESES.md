# Initial spot product hypotheses

Spot products are used to validate payment intent before building subscriptions.

## Candidate products

1. Verified M&A-policy company list
2. ROIC-management disclosure list
3. Shareholder-return policy change list
4. Overseas-expansion strategy list
5. New-business strategy list
6. Human-capital target disclosure list
7. Business-portfolio restructuring list
8. Industry mid-term-plan comparison workbook

## Required fields

- company and security code
- market and industry
- source document and official URL
- source page where available
- publication date and verification date
- relevant structured statement
- quality state and limitations

## Validation criteria

- at least three target users request the same theme;
- the delivered data is materially faster than manual IR research;
- a sample can demonstrate the value without exposing the full product;
- the product can be updated or repeated using the same data model;
- the user understands that it is an information product, not investment advice.
