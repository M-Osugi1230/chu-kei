# 品質ダッシュボード v1

## 目的

掲載企業数ではなく、確認済み範囲・不足項目・レビュー優先順位をユーザーと運営の双方に説明できる状態を作る。

## 文書準備度

詳細抽出済みβ70社を、公式ソース、公表日、ページ証跡、構造化、数値・方針抽出、進捗接続、原文証跡の7項目で機械評価する。

文書準備度は作業順を決める指標であり、本番昇格の自動判定ではない。

## 本番昇格の必須レビュー

1. 原文との数値・単位・年度突合
2. 戦略分類の妥当性確認
3. 別確認者によるダブルチェック
4. 会社詳細、比較、スマホ表示の監査

## 現在の監査結果

- 詳細抽出済みβ: 70社
- 優先度A: 17社（ページ証跡あり）
- 優先度B: 53社（主にページ証跡要確認）
- 本番30社の公表日要確認: 4社
- 本番30社のページ証跡要確認: 23社

未確認項目は推測で補完せず、改善キューとして残す。
