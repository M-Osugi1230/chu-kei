# Production promotion checklist

A company may move from detailed-extracted beta to production only when all applicable checks pass.

- official source identified
- source title confirmed
- publication date confirmed or explicitly unknown
- source verification date recorded
- relevant evidence pages recorded
- financial values verified
- units and fiscal periods verified
- target and actual values distinguished
- strategy classification supported by the source
- capital policy and shareholder-return statements verified
- unknown, undisclosed, and not-extracted states used correctly
- progress records checked for duplicates and orphans
- company detail display checked
- comparison display checked
- mobile display checked
- first reviewer completed
- second reviewer completed
- quality profile recalculated
- release history updated
