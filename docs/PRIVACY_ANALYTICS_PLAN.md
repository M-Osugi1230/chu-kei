# Privacy-conscious analytics plan

Chu-kei should measure product usefulness without collecting unnecessary personal information.

## Events

- company search performed
- strategy filter selected
- comparison opened
- number of companies compared
- official source opened
- zero-result search
- quality explanation opened
- export or notification interest expressed

## Do not collect by default

- sensitive personal information
- full raw search histories tied to an identified person
- document content supplied by users
- cross-site advertising profiles

## Purpose

Analytics is used to improve discovery, comparison, data quality communication, and future paid workflow design. It is not used to infer investment intent or deliver securities recommendations.
