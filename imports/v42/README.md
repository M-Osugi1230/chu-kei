# v42 source import

このディレクトリは、v43再構築の正本となるv42運用パッケージを受け取るための場所です。

## 必要なファイル

`chukei_570_company_operations_v42.zip`

SHA-256は `package.sha256` に固定しています。異なるファイル、破損ファイル、再圧縮されたファイルはインポート処理を失敗させます。

## 自動処理

ZIPがこのブランチへ追加されるとGitHub Actionsが次を実行します。

1. SHA-256照合
2. v42公開パッケージ展開
3. v43日付モデルへの変換
4. 品質プロファイル生成
5. Coverageβの分析プレースホルダー除去
6. 公開サイトを `site/` に生成
7. strict品質ゲート実行
8. 合格した生成物のみブランチへコミット

## 品質原則

- 企業探索570社と主要論点構造化100社を混同しない
- 資料公表日とソース確認日を分離する
- Coverageβを評価済みデータに見せない
- 推測で未確認項目を補完しない
- 投資助言・銘柄推奨を目的としない
