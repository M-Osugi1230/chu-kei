# 公式ソース確認カバレッジ v1

## 目的

Chu-keiでは、掲載企業数とデータの深さを分けて管理する。

- **掲載企業**: JPX上場情報を確認した企業
- **公式ソース確認済み**: 公式IRページへの到達とHTTPS応答を確認した企業
- **主要論点構造化済み**: 中期目標・主要数値・戦略テーマを公式資料から抽出した企業
- **本番品質**: 人手レビューとダブルチェックを完了した企業

## 50%マイルストーン

2026-07-14時点の基準値は次の通り。

- 掲載企業: 570社
- 公式ソース確認済み: 285社（50.0%）
- 主要論点構造化済み: 206社（36.1%）
- 一次確認β: 79社
- Coverageβ: 285社
- 本番品質: 30社

## 一次確認βの契約

一次確認βは、公式IRページの所在確認までを示す。次の情報を確認済みとは扱わない。

- 中期経営計画の対象期間
- 売上・利益・収益性・資本効率・株主還元の目標
- 戦略テーマと施策
- ページ番号付き原文証跡

一次確認βでは、公開リンクを検証済みの公式IRページに限定し、探索中に見つけた資料は詳細抽出前の内部候補として扱う。会社ドメインと一致しない資料候補は公開証跡から除外し、訂正履歴のみを保持する。

## 品質原則

- 未確認情報を推測で補完しない
- 一次確認と詳細抽出を混同しない
- 本番昇格を自動化しない
- 公式ソース、確認日、レビュー履歴を保持する
- カバー率向上によって既存の構造化品質を低下させない
