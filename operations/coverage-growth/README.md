# 掲載企業1,200社・公式ソース298社マイルストーン

## 達成値

- 掲載企業: 1,000社 → 1,200社
- 追加企業: 200社
- 公式ソース確認済み: 285社 → 298社
- 主要論点構造化済み: 206社（変更なし）
- 本番品質: 30社（変更なし）
- 一次確認β: 79社 → 92社
- Coverageβ: 715社 → 902社
- 進捗データ: 149件（変更なし）

## 掲載企業の追加

JPX公式「東証上場銘柄一覧」のうち、Prime・Standard・Growthに上場する内国株式を対象とした。既存未掲載企業から市場・業種を分散して200社を選定している。

- Prime: 107社
- Standard: 73社
- Growth: 20社

新規200社はJPX上場情報のみを確認したCoverageβである。中期経営計画の公式資料、計画期間、目標数値、戦略テーマ、ページ証跡は確認済みとして扱わず、推測による補完を行わない。

## 公式ソース確認の追加

Coverageβから13社について、公式企業サイトまたは公式IRページへの到達とHTTPS応答を確認し、一次確認βへ移行した。

一次確認βでは、公開リンクを検証済みの公式IR入口へ統一する。探索時に発見した決算資料・中期経営計画・統合報告書などは内部証跡候補として保持するが、数値、期間、戦略テーマを詳細抽出済みとは扱わない。

当初は公式ソース確認済み400社を探索目標としたが、既存の自動確認基準で十分な証跡を確認できなかった企業は採用しなかった。確認基準を下げず、今回確実に検証できた13社のみを反映している。

## 品質契約

- 未確認情報を推測で補完しない
- 公式ソース確認と詳細構造化を明確に分離する
- 公式IR入口と資料候補を分離する
- 既存の構造化済み206社、本番品質30社、進捗149件を維持する
- 件数、容量、検索、重複、証跡、ブラウザ表示、アクセシビリティをCIで検査する
