# Quality Score v1

Chu-keiの品質表示は企業評価ではなく、登録データの確認状態を示す。

## 評価対象

- 公式ソース確認
- ページ証跡
- 主要論点の構造化
- 進捗実績の接続
- 人手レビュー
- ダブルチェック

## 品質階層

| 表示 | データ状態 | スコア扱い |
|---|---|---|
| ★★★★★ | 本番品質 | 算定対象 |
| ★★★★☆ / ★★★☆☆ | 詳細抽出済みβ | 算定対象 |
| ★★☆☆☆ | 一次確認β | 暫定値 |
| ★☆☆☆☆ | カバレッジβ | 算定対象外 |

カバレッジβの星はデータ階層を視覚的に示すための表示であり、企業内容の評価点ではない。画面では必ず「算定対象外」と併記する。

## 禁止事項

品質スコアを企業の投資魅力度、経営力、将来性、株価上昇可能性として表示・説明しない。
