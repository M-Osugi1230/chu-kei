# Chu-kei 商用受付データ取扱方針 v1

## 原則

実際の依頼者の氏名、メールアドレス、会社名、案件内容、見積額、成果物はGitHubリポジトリへ保存しない。

## 保存先

- フォーム受付: Netlify Forms
- 案件管理: アクセス制御された非公開の管理手段
- 納品物: 依頼者ごとに限定共有できる保存先

## GitHubに保存してよいもの

- 商品定義
- フォーム項目のSchema
- 空の雛形
- 運用手順
- 匿名化・集計済みの件数
- 個人や案件を特定できない改善指標

## GitHubに保存しないもの

- 実在する氏名、メールアドレス、電話番号
- 依頼者の会社名と具体的な案件内容の組み合わせ
- 見積書、請求書、契約書
- 未公開情報、機密情報
- 個別納品レポート
- Netlify Formsから出力した原本CSV

## 取扱期間

受付情報は、見積・調査・納品・問い合わせ対応に必要な期間に限定して保持する。不要になったデータは削除する。

## リポジトリ保護

`operations/commercial/private/`、`operations/commercial/leads/`、Netlify FormsのCSVエクスポートは`.gitignore`で除外する。誤ってコミットした場合は、通常の削除だけでなく履歴からの除去と認証情報・共有設定の確認を行う。
