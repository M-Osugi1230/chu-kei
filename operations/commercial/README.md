# Commercial operations

このディレクトリには商品定義と運用上の公開可能な設定だけを保存します。

## 保存してよいもの

- `offers-v1.json`: 商品・価格・提供範囲・レビュー条件
- 個人情報を含まない運用ルール
- 匿名化・集計済みの件数

## 保存してはいけないもの

- 実際のフォーム回答
- 氏名、メールアドレス、会社名
- 案件の具体的な依頼内容
- 見積、請求、契約情報
- 個別成果物

実際の受付データはNetlify Formsとアクセス制御された非公開管理手段で扱います。`private/`と`leads/`は`.gitignore`で除外されています。
